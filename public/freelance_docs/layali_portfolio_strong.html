1. نبذة عن المشروع البرمجي

قمت بتطوير منصة "ليالي كشتات" من الصفر، وهي منصة ويب تهدف لتنظيم حجوزات الرحلات البرية. المشروع ليس مجرد تصميم واجهة، بل نظام متكامل يحتوي على لوحة تحكم، قاعدة بيانات، ونظام إدارة محتوى.

2. التقنيات المستخدمة (Technical Stack)
Next.js 14 (Framework) React.js TypeScript Tailwind CSS Supabase (Database) Vercel Deployment

تم بناء المشروع باستخدام هيكلية المجلدات الحديثة (App Router) لضمان سرعة الأداء وتوافقية محركات البحث (SEO).

3. هيكلية المشروع البرمجية (Code Structure)

صورة توضح المجلدات البرمجية الحقيقية للمشروع على بيئة التطوير المحلية (VS Code):

c:\projects\layali-kashtat\ ├── app\ │ ├── [locale]\ # دعم تعدد اللغات (عربي/إنجليزي) │ ├── api\ # واجهات برمجة التطبيقات (Backend APIs) │ ├── dashboard\ # لوحة تحكم الإدارة │ └── layout.tsx # التخطيط العام للصفحات ├── components\ # عناصر الواجهة القابلة لإعادة الاستخدام ├── lib\ │ ├── db.ts # إعدادات الاتصال بقاعدة البيانات │ └── supabaseClient.ts ├── public\ # الملفات والوسائط └── middleware.ts # نظام الحماية والتحويل
